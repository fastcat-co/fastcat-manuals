로그분석기 구동 및 정지
==============

<span></span>
1. 서비스 구동
------------

#### 버전 <= 1.4.2
```
$ analytics/bin/daemon.sh start
```

### 버전 >= 1.4.3
```
$ analytics/bin/analytics.sh start
```


2. 서비스 정지
------------

#### 버전 <= 1.4.2
```
$ analytics/bin/daemon.sh stop
```

### 버전 >= 1.4.3
```
$ analytics/bin/analytics.sh stop
```
